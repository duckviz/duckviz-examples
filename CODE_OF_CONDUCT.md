# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

By participating in this project — opening issues, sending pull requests, joining discussions — you agree to uphold it. The full text is maintained at the link above; we don't keep a copy in this repo so it can't drift out of sync with the canonical version.

## Reporting

Report unacceptable behavior to **vikas@duckviz.com**. Reports are reviewed and acted on confidentially. We won't disclose the reporter's identity without their consent.
